\begin{vers}
Så länge blicken är mild,

Så länge spegel på väggen ger halvskaplig bild, 
Så länge alla kan stå,
Så länge alla kan gå, 
Så länge alla kan nubba så fyller vi på.
\end{vers}
\end{song}
